Guia de Configuração Completo

Conecte o App do celular → Google Sheets → Dashboard em 3 passos

Passo 1 — Preparar a Planilha
1 Importe o modelo para o Google Sheets
Abra o arquivo planilha_senai_modelo.xlsx que foi gerado e importe-o para o Google Sheets:

drive.google.com → Arraste o arquivo → Clique com botão direito → "Abrir com Google Planilhas"

Assim o arquivo vira nativo do Google Sheets e as fórmulas funcionam perfeitamente.
⚠️ Não compartilhe a planilha. Deixe-a como Privado (só você). Apenas o Apps Script precisa de acesso — e ele já tem automaticamente por ser do mesmo Google.
2 Copie o ID da planilha
Na URL da planilha, o ID fica entre /d/ e /edit:

docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms/edit

Guarde esse ID — você vai usá-lo no Apps Script e no Dashboard.
Passo 2 — Configurar o Apps Script
3 Crie o Apps Script
Dentro da planilha:
Menu Extensões → Apps Script

Apague todo o código existente e cole o script abaixo: 
// ══════════════════════════════════════════════════
// SENAI · Apps Script — Receptor de Avaliações
// Cole este código no Google Apps Script da planilha
// ══════════════════════════════════════════════════

var SHEET_ID = "COLE_SEU_ID_AQUI"; // ← substitua pelo seu ID

function doPost(e) {
  try {
    var ss   = SpreadsheetApp.openById(SHEET_ID);
    var data = JSON.parse(e.postData.contents);

    // ── Grava em "Avaliações" ──────────────────────
    var abaAval = ss.getSheetByName("Avaliações");
    if (!abaAval) throw new Error("Aba Avaliações não encontrada");

    var vetor = data.vetor || "";
    var pesoMap = {"Fazer (40%)": 0.4, "Saber (30%)": 0.3, "Comport. (30%)": 0.3};
    var peso = pesoMap[vetor] || 0.3;
    var notaPonderada = (parseFloat(data.nota) * peso).toFixed(2);

    abaAval.appendRow([
      data.data,           // A - Data (dd/mm/aaaa)
      data.turma || "",    // B - Turma
      data.aluno,          // C - Nome do Aluno
      data.uc,             // D - Unidade Curricular
      vetor,               // E - Vetor (Peso)
      data.instrumento,    // F - Instrumento / Atividade
      data.criterio,       // G - Critério Avaliado
      parseFloat(data.nota),// H - Nota
      parseFloat(notaPonderada), // I - Nota Ponderada (calculada)
      data.observacao || "" // J - Observação
    ]);

    // ── Log de timestamp na última coluna ──────────
    var ultimaLinha = abaAval.getLastRow();
    abaAval.getRange(ultimaLinha, 11).setValue(new Date());

    return ContentService
      .createTextOutput(JSON.stringify({status: "ok", linha: ultimaLinha}))
      .setMimeType(ContentService.MimeType.JSON);

  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({status: "erro", msg: err.message}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(e) {
  // Endpoint de health check — confirma que o script está ativo
  return ContentService
    .createTextOutput(JSON.stringify({
      status: "online",
      planilha: SpreadsheetApp.openById(SHEET_ID).getName(),
      timestamp: new Date().toISOString()
    }))
    .setMimeType(ContentService.MimeType.JSON);
}

// ── Função auxiliar: testar manualmente no editor ──
function testarScript() {
  var dados = {
    data: "01/01/2025", turma: "Técnico Mecânica A",
    aluno: "Teste Silva", uc: "Produção Mecânica",
    vetor: "Fazer (40%)", instrumento: "Observação Oficina",
    criterio: "Exatidão Dimensional", nota: 7.5, observacao: "Teste"
  };
  var ss       = SpreadsheetApp.openById(SHEET_ID);
  var abaAval  = ss.getSheetByName("Avaliações");
  abaAval.appendRow([
    dados.data, dados.turma, dados.aluno, dados.uc,
    dados.vetor, dados.instrumento, dados.criterio,
    dados.nota, dados.nota * 0.4, dados.observacao, new Date()
  ]);
  Logger.log("✅ Linha de teste inserida com sucesso!");
}
4 Implante como App da Web
No editor do Apps Script:

1. Clique em Implantar → Nova implantação
2. Tipo: App da Web
3. Executar como: Eu (seu email)
4. Quem tem acesso: Qualquer pessoa
5. Clique em Implantar e copie a URL gerada
ℹ️ "Qualquer pessoa" significa apenas que o App do celular pode enviar dados. Não dá acesso à sua planilha — ela continua privada.
✅ Teste o script antes: no editor, selecione a função testarScript e clique em Executar. Se uma linha aparecer na aba Avaliações, está funcionando.
Passo 3 — Configurar o Dashboard
5 Configure o Dashboard Streamlit
No arquivo dashboard_senai_v2.py, localize a linha: 
SHEET_ID = "SEU_ID_AQUI"
Substitua pelo ID copiado no Passo 2. A planilha precisa estar compartilhada como "Qualquer pessoa com o link pode visualizar" para o Dashboard conseguir ler sem autenticação (leitura pública ≠ edição pública).

Para manter totalmente privado, use a opção abaixo com Service Account.
Opcional — Leitura 100% Privada com Service Account
+ Autenticação via Google Service Account
Se não quiser deixar a planilha pública nem para visualização, instale a biblioteca e use autenticação:

pip install gspread google-auth 
# Adicione ao início do dashboard_senai_v2.py
import gspread
from google.oauth2.service_account import Credentials

def carregar_aba_privada(sheet_id, nome_aba):
    creds = Credentials.from_service_account_file(
        "credenciais.json",  # arquivo baixado do Google Cloud
        scopes=["https://spreadsheets.google.com/feeds",
                "https://www.googleapis.com/auth/drive"]
    )
    cliente  = gspread.authorize(creds)
    planilha = cliente.open_by_key(sheet_id)
    aba      = planilha.worksheet(nome_aba)
    return pd.DataFrame(aba.get_all_records())
Para usar: crie um projeto no Google Cloud Console → ative Google Sheets API → crie uma Service Account → baixe o JSON de credenciais → compartilhe a planilha com o email da service account como Visualizador.
